# Directories

There is a lot of different functions available in Cangjie regarding directories. We only cover
a small portion of them here.
You can find the rest
[here](https://cangjie-lang.cn/en/docs?url=%2F0.53.13%2Flibs%2Fsource_en%2Fstd%2Ffs%2Ffs_package_api%2Ffs_package_classes.html%23prop-info-1)


Open current directory.

Check `if` subDirectory "`Max_is_amazing`" exists.

If it does not exist, create it.


If it does, open it.

Create a file in the subDirectory.

Go over all directories and files strictly under current directory. (non recursive)


Open parent directory.
Iterate over all subDirectories of the parent directory.
